# Administrador de Citas Veterinarias 🐾
Aplicación web para registrar, editar y eliminar citas veterinarias.

## 📸 Preview
<img width="1887" height="1011" alt="image" src="https://github.com/user-attachments/assets/48e37c3d-5fcf-4b52-a8fe-3bbada52e134" />
Este proyecto fue desarrollado como práctica de JavaScript, trabajando con manipulación del DOM, formularios, clases y persistencia de datos mediante IndexedDB.

## 🚀 Funcionalidades
- Registrar nuevas citas
- Editar citas existentes
- Eliminar citas
- Validar campos obligatorios
- Guardar la información en IndexedDB
- Mostrar dinámicamente las citas guardadas

## 🛠 Tecnologías utilizadas
- HTML
- CSS
- JavaScript
- IndexedDB

## 📚 Conceptos practicados
- Manipulación del DOM
- Event listeners
- Objetos y clases
- Validación de formularios
- Métodos de arrays
- IndexedDB
- Transacciones y cursores
